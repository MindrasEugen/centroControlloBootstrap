# Centro Controllo — Dashboard con Bootstrap
Progetto front‑end che implementa una semplice dashboard responsive utilizzando Bootstrap 5, HTML e CSS.
L’obiettivo è esercitarsi nella costruzione di layout strutturati (navbar, sidebar, cards, sezioni) e nell’uso delle utility class di Bootstrap.

## Tecnologie
- HTML5

- CSS3

- Bootstrap 5

## Stato del progetto
Versione iniziale. La struttura verrà ampliata con nuove sezioni e componenti.
